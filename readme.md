## Installation

1. Clone it
```sh
git clone https://github.com/kerogs/KST-3-wallpaper-engine.git
```

2. Install SASS
```sh
npm install -g sass
```

3. It's all good

## What is it ?

The wallpaper simulates a terminal where commands are displayed randomly.
